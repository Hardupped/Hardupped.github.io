## 避坑指南：为什么你的代码没出我意料的结果？

如果你运气不好，跑的时候报错了，大概率是这三点之一，对照检查一下，基本都是秒修复。

### 1. API Key 复制错了
这是最频繁的错误。API Key 以 `sk-` 开头，请确保你没有复制多余的空格，没有漏字符。如果忘记了 Key，去控制台“API Keys”列表重置一个即可。

### 2. `base_url` 写错了
文档上写得很清楚，[千聚api聚合平台](https://www.qianjuai.com/)的 base_url 是 `https://www.qianjuai.com/v1`，**千万不要在最后加 `/`**，也不要用任何其他地址。

错误示范：`https://www.qianjuai.com`（缺了 `/v1`）、`https://www.qianjuai.com/v1/`（多加了斜杠）

### 3. 模型名称写错了
[千聚api聚合平台](https://www.qianjuai.com/)支持的 Qwen 模型非常全，但具体的名称可能和官方不完全一样。你可以登录控制台 -> “模型列表” -> 搜索“Qwen”，可以看到平台定价的准确名称列表。例如：
