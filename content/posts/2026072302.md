## KimiK2 API 调用 Java 示例：两行代码搞定接入

接入[千聚ai聚合平台](https://www.qianjuai.com/)有多简单？真的只是改一行代码的事。下面我以 Kimi K2 模型为例，给你一个完整的 Java 调用示例，用最流行的 OkHttp 库。

java
package com.qianju.kimi;

import okhttp3.*;
import org.json.JSONObject;

import java.io.IOException;

public class KimiK2Example {
    public static void main(String[] args) {
        OkHttpClient client = new OkHttpClient().newBuilder()
                // 关键步骤1：设置超时时间，避免流式响应超时
                .writeTimeout(60, java.util.concurrent.TimeUnit.SECONDS)
                .readTimeout(60, java.util.concurrent.TimeUnit.SECONDS)
                .build();

        // 构建请求体
        JSONObject requestBody = new JSONObject();
        requestBody.put("model", "kimi-k2"); // 指定模型为Kimi K2
        requestBody.put("stream", false);     // 非流式输出

        // 构造消息数组
        org.json.JSONArray messages = new org.json.JSONArray();
        JSONObject userMessage = new JSONObject();
        userMessage.put("role", "user");
        userMessage.put("content", "你好，请用Java写一个可以处理字符串反转的类。");
        messages.put(userMessage);
        requestBody.put("messages", messages);

        // 关键步骤：构建请求，访问千聚统一API端点
        MediaType JSON = MediaType.parse("application/json; charset=utf-8");
        Request request = new Request.Builder()
                .url("https://www.qianjuai.com/v1/chat/completions")
                .method("POST", RequestBody.create(requestBody.toString(), JSON))
                .addHeader("Authorization", "Bearer YOUR_QIANJU_API_KEY") // 替换为你在千聚申请的key
                .addHeader("Content-Type", "application/json")
                .build();

        try {
            Response response = client.newCall(request).execute();
            if (response.isSuccessful()) {
                String responseBody = response.body().string();
                JSONObject jsonResponse = new JSONObject(responseBody);
                System.out.println("AI回复: " + jsonResponse.getJSONArray("choices").getJSONObject(0).getJSONObject("message").getString("content"));
            } else {
                System.out.println("请求失败，状态码: " + response.code());
                System.out.println("错误信息: " + response.body().string());
            }
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}


看见了吗？核心就是两行代码：

1.  `url("https://www.qianjuai.com/v1/chat/completions")` ：把原来的 OpenAI 或官方地址改成这个。
2.  `addHeader("Authorization", "Bearer YOUR_QIANJU_API_KEY")` ：把 API key 换成在[千聚ai聚合平台](https://www.qianjuai.com/)申请的就行。
