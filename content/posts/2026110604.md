### 第三步：发送一个最简单的请求

下面是一个纯 Java 代码示例，使用 HttpClient 直接调用 GPT-5mini：

java
import java.net.URI;
import java.net.http.*;
import java.net.http.HttpRequest.BodyPublishers;

public class Gpt5MiniDemo {
    public static void main(String[] args) throws Exception {
        HttpClient client = HttpClient.newHttpClient();
        
        String requestBody = "{\"model\": \"gpt-5-mini\", \"messages\": [{\"role\": \"user\", \"content\": \"用Java写一个hello world\"}]}";
        
        HttpRequest request = HttpRequest.newBuilder()
            .uri(URI.create("https://www.qianjuai.com/v1/chat/completions"))
            .header("Content-Type", "application/json")
            .header("Authorization", "Bearer sk-你的千聚ApiKey")
            .POST(BodyPublishers.ofString(requestBody))
            .build();
        
        HttpResponse<String> response = client.send(request, HttpResponse.BodyHandlers.ofString());
        System.out.println(response.body());
    }
}


就这样，**三行核心代码（定义模型、设置请求体、发送请求）** 就完成了调用。无论你用的是原生的 Java HttpClient，还是 Spring Boot 的 `RestTemplate`、`WebClient`，甚至 OkHttp，都是同理。
