## 进阶：流式输出示例（官方写得太烂，我重写）

大多数时候，你希望对话像 ChatGPT 那样一个字一个字地显示出来，这就是流式输出。官方文档示例里对 `stream=True` 处理得很粗糙，得到的是一堆 `ChatCompletionChunk` 对象，不是纯文本。

下面是我亲测可用的流式版本：

python
from openai import OpenAI

API_KEY = "sk-你的API Key"
client = OpenAI(
    api_key=API_KEY,
    base_url="https://www.qianjuai.com/v1",
    timeout=120,
)

def stream_chat(prompt):
    response = client.chat.completions.create(
        model="grok-3-mini",
        messages=[{"role": "user", "content": prompt}],
        max_tokens=2048,
        temperature=0.8,
        stream=True,  # 打开流式
    )
    
    print("Grok 3 Mini 说：", end="")
    full_content = ""
    for chunk in response:
        # 关键：只有 content 不为空时才取
        if chunk.choices[0].delta.content is not None:
            content_piece = chunk.choices[0].delta.content
            print(content_piece, end="")
            full_content += content_piece
    print("\n--- 流式输出结束 ---")
    return full_content

if __name__ == "__main__":
    stream_chat("给我写一个三行诗，主题是夏天。")


**这个版本的优势：**

-   **实时打印**：用户能看到 Thinking 过程。
-   **完整保留**：`full_content` 变量里存储了最终的全部文本，可以用于后续处理。
-   **异常容错**：只处理 `delta.content != None` 的分片，避免输出 `None` 值。
