from openai import OpenAI

    base_url="https://www.qianjuai.com/v1",

千聚的接口地址必须写准确，少了一个`v1`或者写错域名都会报404。注意：这里不填Anthropic的Key，只填千聚的Key，否则认证会失败。

### 第三步：调用Claude Sonnet 4.5模型
在同一个文件里加上调用逻辑：
python
response = client.chat.completions.create(
    model="claude-sonnet-4.5",
    messages=[
        {"role": "user", "content": "用Python写一个快速排序函数，加上详细注释"}
    ],
    max_tokens=1024,
    temperature=0.7
)

print(response.choices[0].message.content)

这里最核心的地方是model参数。千聚的Claude Sonnet 4.5模型名是`claude-sonnet-4.5`，不要写成`claude-3-5-sonnet-20241022`那样的官方版本号，否则API找不到模型会报错。

`max_tokens`控制模型输出长度。如果你不确定设多少，建议先给1024或者2048，跑一次看结果再调整。`temperature`设成0.7是比较稳妥的平衡值，太高的温度容易跑偏，太低又可能输出太生硬。

### 第四步：运行并验证结果
直接在终端运行：
bash

如果一切正常，会打印出一段带注释的快速排序代码。如果出错，最常见的原因是：
