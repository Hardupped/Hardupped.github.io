## 完整 Python 接入示例：从安装到运行

下面是一个完整的操作流程，每一步都有说明。

**1. 安装依赖库**

**2. 编写接入代码**

创建一个新文件 `doubao_test.py`：

python
import os
from openai import OpenAI

# 直接获取API Key，建议存在环境变量里
api_key = os.environ.get("QIANJU_API_KEY", "你的千聚API_KEY")
base_url = "https://www.qianjuai.com/v1/"

client = OpenAI(api_key=api_key, base_url=base_url)

def chat_with_doubao(user_input):
    try:
        response = client.chat.completions.create(
            model="doubao-pro-32k",
            messages=[
                {"role": "system", "content": "你是一位友好的助手。"},
                {"role": "user", "content": user_input}
            ],
            stream=False,  # 如果要用流式输出，设为True，然后逐块处理
        )
        return response.choices[0].message.content
    except Exception as e:
        return f"调用失败：{e}"

# 测试
if __name__ == "__main__":
    result = chat_with_doubao("今天天气怎么样？")
    print(result)

如果输出正常的内容，说明接入成功。你可以在控制台看到模型返回的文本。如果你改成 `stream=True`，输出会像打字机一样逐字出现，适合做聊天机器人。

**4. 接入复杂任务：函数调用**

Doubao 同样支持函数调用（Function Calling），这在构建自动化工作流时非常实用。

python
functions = [
    {
        "name": "get_weather",
        "description": "获取某个城市的天气",
        "parameters": {
            "type": "object",
            "properties": {
                "city": {"type": "string", "description": "城市名称"}
            },
            "required": ["city"]
        }
    }
]

response = client.chat.completions.create(
    model="doubao-plus-32k",
    messages=[{"role": "user", "content": "帮我查一下北京的天气"}],
    functions=functions,
    function_call="auto"
)

print(response.choices[0].message)


模型会返回一个 `function_call` 对象，里面包含 `name` 和 `arguments`，你解析后就可以调用自己的 API 了。
