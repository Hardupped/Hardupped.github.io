## 【进阶】让对话持续下去

完成了最基础的“一问一答”后，你肯定想试试让它记住上下文。这其实就是把历史消息都传到消息列表里。

下面是一个支持连续对话的简单示例（不懂没关系，跟着跑一遍就能看到效果）：

python
from openai import OpenAI

client = OpenAI(
    api_key="你的API Key",
    base_url="https://www.qianjuai.com/v1"
)

# 模拟一个对话的历史
messages = [
    {"role": "system", "content": "你是一个聪明的助手，请用英文回答任何问题。"},
    {"role": "user", "content": "Hello, tell me something about Chinese culture."},
]

# 第一次调用，获取回答
response1 = client.chat.completions.create(
    model="gpt-4o-mini",
    messages=messages
)

# 把助手的回答加到消息列表里，作为上下文
messages.append({
    "role": "assistant",
    "content": response1.choices[0].message.content
})

# 再提问，这次模型会结合前面的回答
messages.append({
    "role": "user",
    "content": "Can you give me an example in English? (基于你刚才说的内容举例)"
})

response2 = client.chat.completions.create(
    model="gpt-4o-mini",
    messages=messages
)

print("助手的完整思考：", response2.choices[0].message.content)
