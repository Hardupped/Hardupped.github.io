**2. 核心配置：只需要改 `apiHost` 和 `token`**

import com.theokanning.openai.completion.chat.ChatCompletionRequest;
import com.theokanning.openai.completion.chat.ChatMessage;
import com.theokanning.openai.service.OpenAiService;
import com.theokanning.openai.service.OpenAiService;
import java.time.Duration;

public class ClaudeJavaClient {
    public static void main(String[] args) {
        // 帅炸的操作：只需要把 base_url 换成千聚的 API 域名
        String token = "sk-你的千聚API_KEY"; 
        // 重点！替换成我们的国内直连入口
        OpenAiService service = new OpenAiService(token, Duration.ofSeconds(60));
        service.setApiHost("https://www.qianjuai.com/v1");

        ChatCompletionRequest request = ChatCompletionRequest.builder()
                // 这里直接写模型的完整名字
                .model("claude-3-5-sonnet-20241022")
                .messages(Arrays.asList(
                    new ChatMessage("system", "你是资深Java开发专家"),
                    new ChatMessage("user", "写一段使用CompletableFuture处理高并发订单的代码")
                ))
                .build();

        // 调用并打印结果
        service.createChatCompletion(request).getChoices().forEach(choice ->
            System.out.println(choice.getMessage().getContent())
        );
    }
}


**总结改动点：**
1.  `new OpenAiService(token)` 加上 `setApiHost("https://www.qianjuai.com/v1")`。
2.  把 `model` 名直接写成 Claude 官方的名字（比如 `claude-3-5-sonnet-20241022`）。
3.  把 API Key 换成从千聚拿到的 Key。
