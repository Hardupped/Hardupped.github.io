## Java接入到底有多简单

这一点最关键。我直接给你写一个完整的Java代码样例。

**前提**：你已经注册了千聚，获得了API Key。

1. 导入依赖（如果你用Maven）：

2. 写一个Java方法直接调用o3-mini：

java
import com.theokanning.openai.completion.chat.*;
import com.theokanning.openai.service.OpenAiService;
import java.time.Duration;
import java.util.List;

public class O3MinitDemo {
    public static void main(String[] args) {
        // 就是用千聚的API
        String token = "你的千聚API Key";
        // 核心改动：把base_url换成千聚的
        String baseUrl = "https://www.qianjuai.com/v1";
        
        OpenAiService service = new OpenAiService(token, Duration.ofSeconds(60));
        service.setBaseUrl(baseUrl);
        
        // 构建对话
        ChatCompletionRequest request = ChatCompletionRequest.builder()
                .model("o3-mini") // 直接写模型名
                .messages(List.of(
                    new ChatMessage(ChatMessageRole.USER.value(), 
                    "写一段Java代码，用Stream API计算一个整数列表中每个数的平方根并求和。")
                ))
                .build();
        
        // 发送请求
        ChatCompletionResult result = service.createChatCompletion(request);
        
        // 打印结果
        result.getChoices().forEach(choice -> {
            System.out.println(choice.getMessage().getContent());
        });
    }
}


**就这一个改动**：把`baseUrl`从`https://api.openai.com/v1`改成`https://www.qianjuai.com/v1`，加上你的API Key，代码就能跑通。
