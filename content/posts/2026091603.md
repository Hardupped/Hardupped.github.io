## 接入有多简单？Node.js 示例来了

真的只是改一行代码的事。不管你用 Python 还是 Node.js，只要原来用的是 OpenAI 的 SDK，现在直接把 `baseURL` 改成千聚的 API 地址就行。这里给一个完整的 **Node.js 接入 Kimi K2 对话接口** 的示例：

javascript
// 安装依赖：npm install openai
const OpenAI = require('openai');

const client = new OpenAI({
  baseURL: 'https://www.qianjuai.com/v1', // 关键：改成千聚的地址
  apiKey: 'sk-你的千聚API Key',           // 在千聚后台获取
});

async function chatWithKimiK2() {
  const completion = await client.chat.completions.create({
    model: 'kimi-k2',                     // 直接指定模型名
    messages: [
      { role: 'system', content: '你是一个乐于助人的助手。' },
      { role: 'user', content: '用300字解释什么是量子计算。' },
    ],
  });

  console.log(completion.choices[0].message.content);
}

chatWithKimiK2();


把上面的 `apiKey` 换成你的实际 Key，保存成 `kimi-test.js`，然后在终端运行 `node kimi-test.js`，十几秒后就能看到 Kimi K2 返回的回答。全程不需要代理，不需要海外信用卡。
