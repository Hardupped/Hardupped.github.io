### 第二步：部署代理Sidecar容器

这个容器负责把Mistral的请求转发到千聚API，同时做结果缓存和请求去重。

yaml
apiVersion: apps/v1
kind: Deployment
metadata:
  name: mistral-proxy
spec:
  replicas: 2
  selector:
    matchLabels:
      app: mistral-proxy
  template:
    metadata:
      labels:
        app: mistral-proxy
    spec:
      containers:
      - name: proxy
        image: nginx:1.25
        ports:
        - containerPort: 8080
        env:
        - name: API_KEY
          valueFrom:
            secretKeyRef:
              name: qianju-api-secret
              key: api-key
        - name: BASE_URL
          valueFrom:
            secretKeyRef:
              name: qianju-api-secret
              key: base-url
        volumeMounts:
        - name: cache
          mountPath: /var/cache
      volumes:
      - name: cache
        emptyDir: {}


在Nginx配置里，把`/v1/chat/completions`转发到千聚API，并开启缓存。关键配置如下：

nginx
location /v1/chat/completions {
    proxy_pass https://www.qianjuai.com/v1/chat/completions;
    proxy_set_header Authorization "Bearer $API_KEY";
    
    # 本地缓存：相同请求5分钟内直接返回
    proxy_cache localcache;
    proxy_cache_key "$request_body";
    proxy_cache_valid 200 5m;
    
    # 合并请求：相同请求只发一次上游
    proxy_cache_use_stale error timeout updating;
}
